<h2>프리캠 제작 계기</h2>
<p>프리캠은 2018년 10월부터 개인 독학으로 개발하고 있는 프로그램입니다. <br> 이전 비슷한 계열로 서비스를 제공하는 업체 몇군데를 사용하다<br>자신만의 프로그램이 가지고 싶어 개발하게 되었습니다.</P>
